# SQL Data Analysis - UC Davis Course Project

This project applies SQL concepts learned in the UC Davis SQL for Data Science course to analyze agricultural production datasets.

## Objective

The objective of this project is to practice SQL for data exploration, filtering, aggregation, joins, unions, views, and basic data profiling.

## Dataset

The project uses agricultural production datasets stored in the `data/raw/` folder.

Datasets included:

- Cheese production
- Coffee production
- Egg production
- Honey production
- Milk production
- Yogurt production
- State lookup table
  
## Repository structure

- `data/raw/`: original CSV datasets.
- `docs/`: project documentation, data dictionary, and methodology notes.
- `results/`: exported query results.
- `sql/`: SQL query files.

## SQL concepts used

- SELECT and FROM
- WHERE filtering
- ORDER BY
- GROUP BY
- HAVING
- Aggregate functions: COUNT, SUM, AVG, MIN, MAX
- JOINs
- UNION
- Views
- Basic data profiling

## Project workflow

1. Import the CSV files into a SQLite database.
2. Explore each table individually.
3. Filter and analyze annual production records.
4. Join production data with state information.
5. Combine comparable records using UNION.
6. Export selected query results to the `results/` folder.
